## Installation

To run this application in your local environment, please follow these steps:

```
1. Clone this repository
2. Run 'bundle install'
3. Run 'rails db:migrate'
4. Run 'rails db:seed'
3. Run 'rails s'
4. In your browser, goto http://localhost:3000/
```
